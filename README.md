# Tasks Automation

Tasks Automation is a collection of scripts written in the Powershell language.

## Installation


Clone source code to local host.

```bash
git clone https://github.com/nguyenlethaihoang/tasks-automation.git
```

## Usage

```bash
PowerShell -File [name].ps1
```
